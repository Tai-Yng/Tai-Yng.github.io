cd D:/Bolg/taiyang

## 2. 创建新文章

### 方式一：使用 Hugo 命令（推荐）

```bash
# 进入博客目录
cd D:/Bolg/taiyang

# 创建新文章（会自动生成带 front matter 的 md 文件）
hugo new posts/你的文章标题.md
```

> **提示**：文章标题会自动转换为 URL 友好格式（如 `hello-world`）

### 方式二：手动创建

在 `D:/Bolg/taiyang/content/posts/` 目录下新建 `.md` 文件，参考格式：

```markdown
---
title: "文章标题"
date: 2026-05-14T15:30:00+08:00
draft: false
description: "文章简介"
tags: ["标签1", "标签2"]
categories: ["分类"]
---

这里是文章正文内容...

<!--more-->

摘要分隔符之前的为摘要，会显示在列表页

## 正文

继续写...
```

### 文章模板参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `title` | 文章标题 | `"我的第一篇文章"` |
| `date` | 发布日期 | `2026-05-14T15:30:00+08:00` |
| `draft` | 草稿状态 | `true`=不发布, `false`=发布 |
| `description` | 文章简介 | `"这是一篇关于..."` |
| `tags` | 标签（数组） | `["Hugo", "博客"]` |
| `categories` | 分类（数组） | `["技术"]` |
| `cover` | 封面图片 | `"/images/cover.jpg"` |

## 4. 提交到 GitHub

### 第一次完整提交（新文章）

```bash
cd D:/Bolg/taiyang

# 1. 查看变更
git status

# 2. 添加要提交的文件（新增文章或修改的文件）
git add content/posts/你的文章标题.md

# 或添加所有变更
git add -A

# 3. 提交（替换提交信息）
git commit -m "添加文章：你的文章标题"

# 4. 推送到 GitHub
git push origin main
```

### 后续快速提交流程

```bash
cd D:/Bolg/taiyang
git add -A
git commit -m "更新文章"
git push origin main
```

## 5. 常见问题

### Q1: Git Push 失败（网络连接）

**错误信息**：`Failed to connect to github.com port 443`

**解决方案**：

```bash
# 方法1：配置 Git 全局代理（推荐）
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890

# 方法2：如果没有代理，使用 SSH 方式
# 先检查/生成 SSH key
cat ~/.ssh/id_rsa.pub
# 如果没有，生成：ssh-keygen -t rsa -C "your_email@example.com"

# 然后在 GitHub Settings -> SSH Keys 添加公钥
# 修改 remote 为 SSH 方式
git remote set-url origin git@github.com:Tai-Yng/Tai-Yng.github.io.git
```

> **提示**：将 `127.0.0.1:7890` 替换为你实际的代理地址

### Q2: 想撤销未提交的更改

```bash
# 撤销工作区的修改（谨慎使用！）
git checkout -- content/posts/你的文章标题.md

# 或恢复到最后一次提交的状态
git restore content/posts/你的文章标题.md
```

### Q3: 想查看提交历史

```bash
# 查看最近5次提交
git log --oneline -5

# 查看详细提交信息
git log -p
```

### Q4: 修改了已发布的文章

```bash
cd D:/Bolg/taiyang
git add -A
git commit -m "更新文章：文章标题"
git push origin main
```

### Q5: 想删除一篇文章

```bash
cd D:/Bolg/taiyang

# 删除本地文件
rm content/posts/文章标题.md

# 提交删除
git add -A
git commit -m "删除文章：文章标题"
git push origin main
```
